# Phase 2: ユースケース定義

- **作成日:** 2026-07-15
- **作成ロール:** 壁打ちナビゲーター

登場アクター: **メンバー**（正社員・契約社員）、**アルバイト/パート**、**管理者**（取締役・マネージャー）、**AI社員**（AIネイティブカンパニーの仮想メンバー）、**外注**（稼働報告のみ。労働時間管理の対象外）。

## UC-01 出勤して打刻し、勤怠が自動集計される

- **入力:** ダッシュボードの打刻ウィジェットで「出勤」「休憩開始/終了」「退勤」を押す
- **処理:** 打刻レコード生成（客観的記録・ソース記録）→ 日次集計（所定内/法定内残業/法定外残業/深夜/休日の 6 バケット分解）→ 36 協定アラート判定（月 45h 接近・単月 100h・2〜6 ヶ月平均 80h・45h 超 6 回）
- **出力:** 日次/週次/月次ビュー、月次サマリー、アラート通知
- **例外:** 打刻漏れ・誤打刻 → 修正申請（理由必須）→ 管理者承認 → 修正履歴保全

## UC-02 稟議を申請し、金額に応じた経路で承認される

- **入力:** 稟議区分（購買/契約/経費/採用/出張/その他）・金額・内容・添付
- **処理:** 職務権限マトリクス（区分×金額）から承認経路を自動決定 → 各ステップで承認/却下/差戻し（コメント必須）→ 承認ログ記録
- **出力:** 決裁番号付き稟議、承認待ちタスク通知、監査証跡
- **例外:** 差戻し → 修正して再申請。承認者不在 → 代理承認

## UC-03 日報を提出し、課題が管理者へ暗黙にエスカレーションされる

- **入力:** プロジェクト別の作業内容・工数（0.25h 刻み）・所感・課題・明日の予定
- **処理:** 提出 → 工数集計（勤怠実労働時間との乖離チェック）→ **課題記入や停滞シグナルを検知すると管理者へエスカレーション起票**（本人の申告負担なし = 暗黙の情報共有）→ 管理者は「回答送信 / 裁定記録 / 対応不要」のアクションを実行 → 裁定はナレッジへ還流
- **出力:** 日報一覧・提出状況・コメント/リアクション・エスカレーションセンター
- **例外:** 未提出 → リマインド通知

## UC-04 アルバイトがシフト希望を出し、管理者が調整・確定する

- **入力:** 募集期間に対する勤務希望（出勤可/NG/どちらでも + 時間帯）
- **処理:** 締切 → 週別グリッドで必要人数と突合・調整（休憩不足・18 歳未満深夜・週 40h 超をバリデーション）→ 確定・公開 → 確定後変更は本人合意ステップ
- **出力:** 確定シフト（本人はモバイルカード表示）、人員過不足ビュー

## UC-05 AI社員にタスクを依頼し、活動を管理する

- **入力:** 3D オフィス空間の AI社員（ロールは設定で自由に作成・編集）を選んでタスクを依頼
- **処理:** AI がタスクを分解 → 依頼者が承認 → 実行（活動ログを逐次記録）→ 完了報告。確信度が低い場合は管理者へエスカレーション
- **出力:** タスクボード、活動ログ、**日次活動報告（日報/週報の場所に AI社員の報告として掲載）**

## UC-06 管理者が意思決定支援ツールで事業判断を行う

- **入力:** 判断テーマ（例: 停滞プロジェクトの立て直し、提供システムの値上げ、採用可否）
- **処理:** 対象の①意味（属性・KPI）②関係（顧客・PJ・メンバーとの繋がり）③制約（予算・契約・稼働）を提示 → 制約で潰れる打ち手を可視化 → 実行可能な選択肢 A/B/C + AI 推奨 → 判断を記録
- **出力:** 意思決定ログ（後から根拠を追跡可能）

## UC-07 提供システムの稼働状況を確認する

- **入力:** ダッシュボードの稼働状況サマリをクリック
- **処理:** システム別の状態（正常/性能低下/一部障害/重大障害/メンテ）を最悪値ロールアップ、90 日稼働率、インシデント履歴（調査中→原因特定→経過観察→解決）
- **出力:** ステータス詳細ページ

## UC-08 マスタとナレッジを整備し、AI の文脈になる

- **入力:** メンバー/業界/自社/顧客(会社)/顧客(人)/顧客関係/プロジェクトの各マスタ、5 ドメインのナレッジ
- **処理:** CRUD + カスタム項目（任意設定）+ 関係の有向エッジ管理
- **出力:** チャットボット・意思決定支援・AI社員が参照する文脈。分析用ディメンションの給源

## システム化の仕分け

| 対象 | 内 / 外 |
|---|---|
| 打刻・勤怠集計・有給管理・シフト・日報・稟議・マスタ・ナレッジ・AI機能 | システム内 |
| 給与計算・労基署届出・契約書リーガルチェック | システム外（データ出力で連携） |
| サポート管理表等の既存外部ツール | 外部リンクとして統合（設定可能） |
| 外注の労働時間管理 | 対象外（偽装請負リスク回避。稼働報告のみ） |
